# GTM Debugger

GTM Debugger는 Google Tag Manager의 dataLayer 이벤트를 실시간으로 추적하고 시각화하는 크롬 확장 프로그램입니다.

## 주요 기능

- 브라우저에서 발생하는 window.dataLayer 객체에 변화를 감지합니다
- dataLayer 객체에 변화를 시각화 합니다
